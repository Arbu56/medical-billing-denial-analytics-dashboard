# Medical-billing-denial-analytics-dashboard

## Project Overview
This project analyzes healthcare medical billing and denial management data to identify denial trends, payer performance, provider efficiency, and revenue cycle insights using Python and Power BI.
The project focuses on transforming raw healthcare claims data into actionable business insights through data cleaning, exploratory data analysis (EDA), KPI generation, and interactive dashboard visualization.## Project Overview
This project analyzes healthcare medical billing and denial management data to identify denial trends, payer performance, provider efficiency, and revenue cycle insights using Python and Power BI.
The project focuses on transforming raw healthcare claims data into actionable business insights through data cleaning, exploratory data analysis (EDA), KPI generation, and interactive dashboard visualization.

## Tools & Technologies Used
- Python
- Pandas
- NumPy
- Excel
- Power BI

## Business Insight and Trends
- Analyze healthcare claim denial trends
- Identify top denial codes and denial categories
- Evaluate payer-wise denial performance
- Analyze provider-wise revenue and denials
- Monitor financial KPIs and revenue cycle metrics
- Improve operational visibility through dashboard reporting

## Data Cleaning & Preparation

The following preprocessing steps were performed using Python:
- Removed duplicate records
- Standardized date formats
- Handled missing/null values
- Renamed inconsistent columns
- Created analytical feature columns
- Generated denial and paid flags
- Prepared dashboard-ready dataset

## Key KPIs
- Total Claims
- Total Billed Amount
- Total Paid Amount
- Denied Amount
- Adjusted Amount
- Denial Rate
- Provider Enrollment Denials
- Insurance-wise Denials
- Provider-wise Revenue

## Key Insights
- CO18 duplicate claim denials contributed the highest denial volume.
- Molina had the highest denied dollar amount among insurance providers.
- Denial rate reached approximately 48%, indicating significant revenue cycle inefficiencies.
- Provider enrollment denials negatively impacted collections.
- Dr. Wilson generated the highest billed revenue amount.
